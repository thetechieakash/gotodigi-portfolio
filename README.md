# 🚀 GotoDigi | Next-Gen Digital Content Platform

**GotoDigi** is a high-performance, modern web application designed for interactive digital experiences and rich content creation. Powered by **React 19**, **Appwrite**, and **Tailwind CSS v4**, GotoDigi combines robust global state management, smooth hardware-accelerated animations, and a powerful collaborative rich-text editing experience.

---

## ✨ Key Features

- **🌐 Advanced Rich Text Engine:** Complete integration with **TipTap Starter Kit** including code highlights (`lowlight`), underlines, image alignment, hyperlinks, and custom placeholders.
- **⚡ Supercharged Backend:** Fully integrated with **Appwrite v26** handling authentication, database management, and scalable storage securely.
- **🎨 Dynamic & Fluid UI:** Styled beautifully using the brand-new **Tailwind CSS v4** engine and brought to life with high-fidelity animations via **Motion**.
- **🧠 Reliable State Management:** Centralized data store built on top of **Redux Toolkit** and **React Redux** for predictable rendering cycles.
- **🧭 Client-Side Routing:** Lightning-fast navigation throughout the platform via **React Router DOM**.
- **📝 Form Management:** Form state handling and inputs validation powered cleanly by **React Hook Form**.

---

## 🛠️ The Tech Stack

### Core Framework & Build Tooling
*   **React 19.2.4** - Component-driven architecture using the latest concurrent rendering features.
*   **Vite 8.0.1** - Next-generation frontend tooling providing ultra-fast HMR (Hot Module Replacement).

### Backend-as-a-Service (BaaS)
*   **Appwrite 26.0.0** - Secure User Management, Databases, and Storage Buckets.

### Rich Text Editor Suite (TipTap & Quill)
*   `@tiptap/react` & `@tiptap/starter-kit` - Fully customizable headless text editor framework.
*   `@tiptap/extension-code-block-lowlight` - Syntactically beautiful code blocks.
*   `html-react-parser` - Safeguarded rendering of dynamic HTML strings into pure React nodes.

### State & Networking
*   **Redux Toolkit (`@reduxjs/toolkit`)** - Industry standard state workflow management.
*   **Axios** - Promise-based HTTP client for secure external endpoint communication.

## 📦 Project Setup & Installation

### Prerequisites
Make sure you have [Node.js](https://nodejs.org/) installed on your machine.

### 1. Clone the Repository
```bash
git clone https://github.com/thetechieakash/gotodigi-portfolio
cd gotodigi-portfolio
```

### 2. Install Dependencies
```bash
npm install
```

### 3. Setup Environment Variables
Create a `.env` file in the root directory and connect your Appwrite services:
```env
VITE_APPWRITE_URL=appwrite_end_point
VITE_APPWRITE_PROJECT_ID=your_project_id
VITE_APPWRITE_DATABASE_ID=your_database_id
VITE_APPWRITE_COLLECTION_ID=your_collection_id
VITE_APPWRITE_BUCKET_ID=your_bucket_id
```

### 4. Run the Development Server
```bash
npm run dev
```
Open [http://localhost:5173](http://localhost:5173) in your browser to view the application.
